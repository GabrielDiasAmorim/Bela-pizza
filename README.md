# 🍕 Bella Pizzaria - Site

Este é um projeto de site para uma pizzaria fictícia chamada **Bella Pizzaria**, desenvolvido com foco em design moderno, responsividade e experiência do usuário.

🔗 Acesse o projeto:
👉 https://seu-link-aqui.com

---

## 📌 Sobre o projeto

O site apresenta um cardápio digital interativo com:

* 🍕 Pizzas com imagens reais
* 💰 Preços por tamanho (Pequena, Média e Grande)
* 🥤 Sessão de bebidas
* 📱 Layout responsivo (funciona em celular e desktop)
* 🎨 Design inspirado em pizzarias italianas

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts
* Imagens externas (Unsplash)

---

## 🎨 Design

O layout foi desenvolvido com base nas cores da identidade visual da pizzaria:

* 🔴 Vermelho (destaque / marca)
* ⚪ Branco (contraste)
* ⚫ Preto/Cinza escuro (fundo elegante)

Fontes utilizadas:

* Playfair Display (títulos)
* Poppins (textos)
